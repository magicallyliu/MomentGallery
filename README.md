# 须臾图库项目

------

## 项目介绍

基于Vue 3 + Spring Boot + COS + WebSocket 的智能协同云图库平台



本项目分为三个模块：公共素材空间，用户私人空间，团队空间。

1）公共素材空间：用户可以根据自己的需求，在网站中找到需要的图片素材，同时自己也可以上传图片到公共素材空间，只需要经过管理员审核，就可以展示给其他用户。
